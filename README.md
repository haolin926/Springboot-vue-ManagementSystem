# spring-vue-demo
## Overview
This is a employee management system base on **Java SpringBoot framework**, **MyBatis** and **Vue.js**,
- Allows user to view and manage data of employees and departments (Add, Edit, Delete)

>---
> ## Tools used
> - Front-end
>    - Vue.js
>    - Axios
>    - element-ui
> - Back-end
>   - Java
>   - Springboot Framework
>   - MyBatis Framework
>   - MYSQL
> ---
---
